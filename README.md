# Image_classifier
AI Image Detective: Multi-Head Classifier. This project provides an end-to-end pipeline for building a deep learning model that performs dual-task classification. It identifies the subject of an image (e.g., cat, dog) while simultaneously detecting its source (Real vs. AI-generated). 
The system consists of a preprocessing utility for dataset labelling and a TensorFlow-powered multi-head neural network.ðŸš€ Features: Automated Labelling: A script to bulk-rename AI-generated images with specific suffixes to prepare them for supervised learning. Multi-Head Architecture: Uses a shared MobileNetV2 backbone with two specialised output branches: Subject Head: Multi-class classification for identifying the object.
Source Head: Binary classification for AI vs. Real detection.Efficient Pipeline: Custom tf.data pipeline that extracts labels directly from directory structures and filenames on the fly.
